# 결(GYEOL)

현재 내 위치를 1861년 대동여지도의 기호, 길, 옛지명으로 번역하는 공공 문화데이터 기반 AI 문화유산 해독 엔진입니다.

## 프로젝트 요약

결(GYEOL)은 대동여지도를 단순히 크게 보여주는 서비스가 아닙니다. 사용자가 "제가 지금 있는 곳은 대동여지도에서 어디였나요?"라고 물으면 현재 위치를 옛 지명 후보와 연결하고, 실제 판본 이미지와 공공 문화데이터 근거를 바탕으로 답변하는 서비스입니다.

핵심 전환은 다음과 같습니다.

- 보는 지도에서 묻고 답하는 지도로
- 감상하는 지도에서 해독하는 지도로
- 옛 지도에서 내 위치와 연결되는 지도로

## 기획상 핵심 데모

질문:

> 제가 지금 있는 곳은 대동여지도에서 어디였나요?

입력 예시:

- 현재 위치: 천안역

처리 흐름:

1. 현재 위치 확인
2. 현대 행정구역과 주변 POI 변환
3. 옛지명 후보 3개 추출
4. 대동여지도 해당 영역과 교차검증
5. 신뢰도 산출
6. 판본 근거, 공공데이터 근거, 추정 라벨과 함께 음성 답변

답변 예시:

> 지금의 천안역은 대동여지도 기준으로 천안군 권역 후보로 읽힐 가능성이 높습니다. 다만 정확한 한 점이 아니라 옛 행정권역 기준의 추정입니다.

중요한 원칙은 단정하지 않는 것입니다. 결은 "정확히 여기다"라고 말하지 않고, 근거와 신뢰도를 함께 제시하며 권역 후보로 추정합니다.

## 현재 프로토타입

현재 구현된 웹 프로토타입은 제공된 기준 화면의 대동여지도 장면과 고지도 자산을 중심으로 구성되어 있습니다.

- 헤더·하단 진행 단계가 고정된 사진형 작업대 위에, 배경 광원과 한지 톤을 맞춘 `daedongyeojido-idle-toned-v2.png` 대동여지도로 시작
- `0.00–1.55s`: 중앙 지도 위 한성부 선택 후 사진형 오프닝 한지는 고정된 채 사라지고 지도 종이만 좌측으로 이동한다. 일반 지도와 먹색 합성본이 교차하고 22px 페더 마스크를 거쳐 이동 완료 배경에 자연스럽게 융화
- `1.64–4.30s`: 먹길을 분리한 `gyeol-folded-route-base-v5.png`의 실제 주름 8면이 순차 전개되고 마지막 면까지 안착
- `3.90–6.25s`: 두 원본 지도에서 분리한 실제 먹선과 인접 원본 픽셀이 부드러운 마스크를 따라 좌측에서 한성부 방향으로 번짐
- `4.50–5.85s`: 모든 접이 면이 펼쳐진 뒤 보정된 먹길 접점을 확대 원점으로 삼아 우측 한성부 상세도 노출
- `6.42–18.92s`: 63개 실측 먹길 좌표와 한성부 시가지 길을 따라 김정호가 이동
- 이동 건너뛰기와 처음부터 다시 보기

실행:

```bash
npm install
npm run dev -- --port 5187
```

접속:

```text

```

## 향후 활용 데이터

결은 공공 문화데이터를 답변 근거로 사용합니다.

- 국립중앙박물관 e뮤지엄 소장품 데이터
  - 대동여지도 신수19997 실제 판본 이미지
  - 소장품 메타데이터
  - 판본 근거 라벨
- 국가유산청 고지도·지도표 해설
  - 지도표 14개 항목, 22종 기호 해설
  - 도로, 읍치, 역참, 봉수, 능침 등 해독
- 문화공공데이터광장 Open API
  - 박물관 및 문화자원 메타데이터 보강
  - 공공데이터 근거 라벨
- 자체 전국 POI 데이터
  - 현재 위치 주변 맥락 보조
  - 옛지명 후보 매칭 보조

## GYEOL Engine

결은 재사용 가능한 엔진 구조를 목표로 합니다.

- VIEW: 실제 대동여지도 판본 이미지를 보여주고 필요한 부분을 레이어로 점진 공개합니다.
- VOICE: 사용자의 음성 질문을 받아 STT, RAG, TTS 구조로 답변합니다.
- ASK: 사용자가 자연어로 자유롭게 질문합니다.
- GEO: 옛지도와 현대 좌표를 연결하고 후보 3개와 신뢰도를 제공합니다.
- DATA: 국립중앙박물관, 국가유산청, 문화공공데이터 등 공공 문화데이터를 정규화합니다.

## 발표 메시지

심사위원이 30초 안에 이해해야 하는 문장은 다음입니다.

> 결은 대동여지도를 크게 보여주는 서비스가 아니라, 지금 내 위치를 1861년 대동여지도의 기호, 길, 지명으로 번역하는 공공 문화데이터 해독 엔진입니다.

발표에서 반복할 키워드는 다음 5개입니다.

- 현재 위치
- 옛지명 후보
- 대동여지도 실제 판본
- 판본 근거 / 공공데이터 근거 / 추정
- 묻기만 하면 읽히는 문화유산

## 표현 원칙

피해야 할 표현:

- AI가 대동여지도를 복원한다
- 천안역은 정확히 대동여지도에서 천안군이다
- 모든 국가유산에 바로 적용 가능하다
- 대동여지도를 새로 그린다

사용해야 할 표현:

- 실제 판본 이미지를 기반으로 해독한다
- 권역 단위 후보로 추정한다
- 고지도·고문헌형 유산부터 확장한다
- 판본 근거, 공공데이터 근거, 추정을 분리해서 보여준다
