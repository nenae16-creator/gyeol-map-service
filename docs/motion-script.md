# GYEOL 한성부 여정 모션 스크립트

## Goal

첫 화면은 사진형 작업대의 넓은 한지 위에 대동여지도 한 장을 보여준다. 좌우에는 벼루·붓·두루마리·고서가 자연스럽게 걸쳐져 빈 공간을 채운다. 사용자가 지도 위 한성부를 선택하면 작업대는 움직이지 않고 첫 화면 한지는 녹아 사라지며 지도 종이만 왼쪽으로 이동해 원래 최종 시안의 좌측 배치에 정착한다. 그 뒤 접이 지도가 완전히 펼쳐지고, 마지막 단계에서만 우측 한성부 상세 지도가 확대되어 나타난다.

## Layer Order

1. 고정 빈 책상판
2. 첫 화면 사진형 한지·작업 소품 배경판
3. 이동 완료 상태의 좌측 지도 배경판
4. 중앙에서 좌측으로 이동하는 대동여지도 종이
5. 마지막에 나타나는 우측 공식 도성도와 선택 장소 추정점
6. 좌측 접이 지도 8개 면과 김정호
7. 실제 먹선 번짐 레이어와 도착 표시
8. 한성부 선택·건너뛰기·다시 보기 컨트롤

## Timeline

- `0.00s`: 중앙 대동여지도와 한성부 선택점을 보여준다.
- `0.00s ~ 1.55s`: 톤 보정 대동여지도 비율을 유지한 채 왼쪽으로 이동·축소하고 3D 회전한다. 첫 화면 사진형 한지판은 지도와 함께 움직이지 않고 `0.45–1.45s`에 흐림과 함께 사라진다. `0.42s`부터 같은 지도의 먹색 합성본을 겹치고 일반 지도는 `0.55s`부터 흐리며, 이동 완료 배경판은 `0.58s`부터 나타난다. 먹색 합성본은 `0.80–1.55s`에 한지 색과 명암으로 녹아들어 지도와 배경이 따로 교체되는 인상을 줄인다.
- `1.64s ~ 3.72s`: 접이 지도 8개 면이 실제 주름 순서대로 `power4.out` 가속과 0.18초 간격으로 독립적으로 펼쳐진다.
- `3.72s ~ 4.30s`: 펼쳐진 면이 짧게 안착하고, 보정된 단일 접이 지도 이미지로 부드럽게 교차 전환한다. 마지막 패널이 사라지는 `4.30s`까지 상세 지도는 노출하지 않는다.
- `3.90s ~ 6.25s`: 흐릿한 선단을 가진 마스크가 길이 제거된 기본판 위로 원본 먹선 픽셀을 좌측에서 우측으로 복원한다. 접이 지도 위에서 시작한 먹이 연결점을 지나 상세 지도 안으로 번진다.
- `4.50s ~ 5.85s`: 먹길 끝에서 국립중앙박물관 소장 대동여지도의 도성도가 흐림을 걷어내며 확대된다.
- `3.90s ~ 6.25s`: 배경 픽셀이 없는 SVG 먹길이 실제 벡터 길이에 따라 점진적으로 그려진다.
- `6.42s ~ 18.92s`: 김정호 캐릭터가 서비스가 구성한 경로를 따라 선택한 도성도 위치 후보로 이동한다. 이는 실제 답사 경로 재현이 아니다.
- `18.94s`: 한성부 중심의 도착 표시가 등장하기 시작하고 `19.00s`에 다시 보기 버튼이 나타나기 시작한다. 최종 포커스 이동은 `19.32s`에 실행된다.

## Placement

1487×1058 기준 좌표를 비율로 사용한다.

- 중앙 지도 종이: 약 `(510,127)–(1022,894)`.
- 중앙 한성부 클릭점: 약 `(689,465)` = `(46.34%,43.95%)`.
- 이동 완료 좌측 지도 배경: `(43,111)–(815,928)`.
- 좌측 접이 지도: `(0,362)–(872,880)`.
- 우측 상세 지도: `(650,121)–(1487,931)`.
- 접이 지도와 상세 지도 먹길 접점: 약 `(872,602)`. 원본 v4에서 분리한 먹선 오버레이와 보행 경로를 가시 종이 영역 기준 `y≈0.463`으로 보정했으며, 상세 지도 확대 원점도 같은 접점인 `25.3% 59.4%`를 사용한다.

## Interaction

- 가운데 지도 위 `한성부`를 누르면 여정이 시작된다.
- 진행 중 `이동 건너뛰기`를 누르면 즉시 한성부 도착 상태로 이동한다.
- 도착 후 `처음부터 다시 보기`를 누르면 중앙 지도 첫 화면으로 돌아간다.
- `prefers-reduced-motion` 환경에서는 이동·전개·보행을 생략하고 최종 장면을 바로 보여준다.
- 첫 화면 사진형 한지판·고정판·페더 마스크·지도·먹선·인물과 여덟 장의 패널 텍스처까지 17개 핵심 이미지가 모두 준비된 뒤에만 한성부 선택을 활성화한다.
- 이미지 로드에 실패하면 한성부 선택을 비활성화하고 새로고침 안내를 상태 영역에 전달한다.
